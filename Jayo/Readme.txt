Product Name
---------------------
Jayo - A Freelancers & Agencies HTML Template

---------------------
Product Description
---------------------
Jayo is a lightweight & minimal HTML template perfect for agencies and freelancers.

Jayo is built with the popular framework Bootstrap 3.x, HTML5, CSS3, jQuery. This template is fully responsive and works on all devices.

---------------------
Product Features
---------------------
Fully Responsive
Page Header with background image
Slider Page Header
Grid Portfolio Layout
Masonry Portfolio Layout
2 Columns Portfolio
3 Columns Portfolio
4 Columns Portfolio
Masonry Blog Layout
Grid Blog Layout
2 Columns Blog
3 Columns Blog
Video support for portfolio items
Video support for posts
About Us Template with team grid, services & logo carousel.
About Me Template with sliding skill bars, education section & work experience section.
Infinite Scroll
And much more...
