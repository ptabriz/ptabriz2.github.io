CSS Files

All css files are located in /css, this template also has an option for sass styles.

  • bootstrap.min.css - Bootstrap framework
  • font-awesome.min.css - Icon library
  • fontello.css - Icon library
  • main.css - Main css file containing all template styles
  • owl.carousel.min.css - Slider styles
  • owl.theme.default.min.css - Slider styles

JS Files

All js files are located in /js.

  • bootstrap.min.js - Bootstrap framework
  • imagesloaded.pkgd.min.js - Detect when images have been loaded
  • isotope.pkgd.min.js - Portfolio grid system
  • jayo-main.js - Main JS file
  • jquery-3.2.1.min.js - jQuery
  • jquery-migrate-1.4.1.min.js - jQuery
  • jquery.appear.js - Detect when element is visible
  • jquery.infinitescroll.js - Infinite Scroll
  • owl.carousel.min.js - Slider

Images

All images are located in /img.

  • img/headers - Header images
  • img/logos - Logos for clients carousel
  • img/portfolio - Images for portfolio & blog
  • img/team - Images for team grid

Portfolio Filter

Use the data-filter="" attribute to connect portfolio items with the portolio filter.

Example: Add <a href="javascript:void(0)" data-filter=".example">example</a> to the filter and add a class named "example" to the portfolio item.

Infinite Scroll

Every blog file and portfolio file comes with a -2.html extension, this is used to setup infinite scroll.
To disable infinite scroll simply remove the navigation class(see image below).

Example: Add portfolio items to portfolio-2-columns.html and add the items you want to load using infinite scroll inside portfolio-2-columns-2.html.

Contact Form

Edit the contact form in contact-form.php.
Please note that you might have to configure the php.ini file on your server to get the form to work(read tutorial)
Example: Change email adress on line 5.
